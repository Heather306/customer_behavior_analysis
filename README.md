# 📊 Data Analytics Project

## Overview

This project demonstrates a complete data analytics workflow, from data preparation to dashboard development and reporting. It includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, interactive dashboard creation in Power BI, and presentation of findings using Gamma.

---

## Dataset

The dataset contains structured records used for analyzing trends, patterns, and key performance metrics. It was cleaned and transformed to ensure accurate analysis and visualization.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data types and dataset structure.

### 2. Exploratory Data Analysis (EDA)

* Examined data distributions.
* Identified missing values and outliers.
* Explored relationships between variables.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into a SQL database.
* Wrote SQL queries to answer business questions.
* Used filtering, grouping, joins, aggregate functions, and sorting to generate insights.

### 5. Power BI Dashboard

* Connected Power BI to the processed dataset.
* Created interactive charts and KPIs.
* Built dashboards for data exploration and decision-making.

### 6. Report & Presentation

* Summarized findings in a project report.
* Created a presentation using Gamma to communicate key insights and recommendations.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter notebooks to run the data loading, EDA, and cleaning processes.
4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
5. Execute the SQL queries in the `sql` folder.
6. Open the Power BI dashboard (`.pbix`) to explore the visualizations.
7. Review the report and presentation for the complete project findings.
